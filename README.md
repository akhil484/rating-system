# Review a Product
A real-time product review app where ratings update instantly across all open tabs, built using FastAPI, React, WebSocket,s and PostgreSQL

## Features
1. Products List
  > - All Products List
  > - Links to Individual Product Page
2. Product Page
  > - Product Name
>   - All Reviews
>   - Reviews include star rating(including decimal reviews) and comments
3. Review Submission Form
  > - Submit Review
>   - Review gets updated in real time
